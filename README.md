# Mintable Token

[![License](https://img.shields.io/badge/License-Apache%202.0-blue.svg)](https://opensource.org/licenses/Apache-2.0)

## Usage

Install dependencies

```
$ npm i
```

Compile contracts

```
$ npm run build
```

Compile contracts and run tests

```
$ npm run test
```

Report test coverage report

```
$ npm run coverage
```
